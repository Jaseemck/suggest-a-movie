Movie Recommendation

This system uses Content Based Recommendations approach to find similar movies. This is a naive method to recommend items similar to some items. The collaborative filtering method is more efficient for movie recommendation and used more often for this very application. The dataset used for this model is TMDB-movie-metadata The dataset is taken from Kaggle.com.
